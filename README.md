
# 🤖 IdeaScribe AI

IdeaScribe AI is a simple text generation web app built using Streamlit and GPT-2.

## 🚀 Features
- Generate text from any prompt
- Adjustable creativity (temperature)
- Control output length
- Clean UI using Streamlit

## 🛠️ Tech Stack
- Python
- Streamlit
- Hugging Face Transformers (GPT-2)

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
